# 📸 AI Studio - Unsplash Image Search App

Welcome to the **Unsplash Image Search App**! This is a clean, minimal, and fast web application that allows users to search and discover high-quality images using the official Unsplash API.

---

## 🚀 Features

* 🔍 **Real-time Image Search:** Search millions of high-resolution photos instantly.
* ⚡ **Fast & Responsive:** Beautifully optimized for both desktop and mobile screens.
* 🎨 **Minimal UI:** Clean aesthetic focusing purely on imagery and seamless user experience.
* 🛠️ **Powered by Unsplash API:** Fetches accurate, live data directly from the official engine.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid)
* **Scripting:** JavaScript (ES6+ Fetch API)
* **API Engine:** Official Unsplash Developers API
